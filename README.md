# Strive

- Running the program
    - Open CSCI338FinalProject.Server folder in a C# IDE such as Visual Studio or Visual Studio Code
    - Build and run the program
    - Navigate to command line
    - Enter `npm install` to install the Node Package Manager
    - Navigate to CSCI338FinalProject.Server/web-portal/
    - Enter `ng serve` to build and serve the application
    - Open the link given in the command prompt window
  
- All data that is returned is in JSON format
